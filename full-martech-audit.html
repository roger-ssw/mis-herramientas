All-in-One Audit

This script runs all 4 audit tiers simultaneously and produces a unified report. One paste in the console, full martech stack analysis. Export as JSON or shareable HTML report.

Instructions:

  1. Open the target site in your browser
  2. Open DevTools (F12 → Console)
  3. Copy the script below (or use the bookmarklet from the home page)
  4. Paste and press Enter
  5. Navigate the site (product pages, cart) to capture more events
  6. Run downloadFullAuditHTML() for a shareable report

    if (window.dataLayer) {

        window.dataLayer.forEach(function(item) {
            if (item.event) {
                if (!dlEvents[item.event]) dlEvents[item.event] = 0;
                dlEvents[item.event]++;

            if (item.ecommerce || (item.event && ['view_item', 'view_item_list', 'add_to_cart', 'begin_checkout', 'purchase'].indexOf(item.event) !== -1)) {
